## Publicar en Vercel

1. Sube estos archivos a un repositorio de GitHub.
2. En Vercel, crea un nuevo proyecto desde ese repositorio.
3. No hace falta configurar framework ni build command.
4. Publica el proyecto.

## Ajustes finales recomendados

- Cambiar `https://elecom-instalaciones.vercel.app/` por la URL real cuando Vercel la genere.
- Sustituir las imágenes orientativas por fotografías reales de trabajos de ELECOM.
- Añadir el logo oficial si está disponible.
